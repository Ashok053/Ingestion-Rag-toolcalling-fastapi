# AI Interview Booking & Conversational RAG

A FastAPI project that allows users to:
- Book interview slots via chat or directly
- Ask questions using a Retrieval-Augmented Generation (RAG) system
- Maintain conversation history using Redis

## Tech Stack
- Python 3.12
- FastAPI
- SQLAlchemy + SQLite
- Redis
- Qdrant
- Groq / Free LLM (for chat RAG)
